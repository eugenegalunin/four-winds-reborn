# Preserved Classic Reference Material

This directory contains original or demo-era Rune War documents retained as
historical evidence. They are not the current Four Winds Reborn manual and
must not silently override implemented rules, tests or the decisions recorded
in [`../../RulesDecisions.md`](../../RulesDecisions.md).

## Inventory

- `Backgrnd.txt` — original Isle of Four Winds background manuscript.
- `Chars.txt` — original wizard biographies and stories.
- `Manual.txt` — English documentation for the version 1.2 demo.
- `Manual.rus` — surviving Russian manual.
- `Manual2.rus` — additional surviving Russian manual fragment.
- `ReadMe.txt` — original distribution readme and file list.
- `Refsheet.txt` — short English reference sheet.
- `mahjong_rules.pdf` — preserved external Mahjong rules reference.

The `.rus` files use legacy encodings and may display incorrectly in a modern
UTF-8 terminal. Preserve them byte-for-byte; do not normalize or re-save them
merely to improve terminal rendering.

Original names, values, spelling and chronology are kept as evidence. Current
behaviour is documented in
[`../../BattleAndAdventureRules.md`](../../BattleAndAdventureRules.md),
[`../../SpellLifecycle.md`](../../SpellLifecycle.md) and the focused tests. If
the sources disagree, the conflict should be made explicit in
[`../../RulesDecisions.md`](../../RulesDecisions.md).
