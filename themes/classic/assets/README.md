# Classic runtime assets

This directory contains the player-ready resources for the restored Classic
theme. Resources are grouped by purpose while retaining their historical file
names, because game data addresses them by basename:

- `images/` — sprites, screens and interface artwork;
- `audio/` — voices and sound effects;
- `music/` — MIDI music;
- `fonts/` — bundled fonts.

`License.txt` records the provenance and distribution terms inherited from the
original project. The theme directory was renamed from `default` to `classic`;
its save and replay content identity intentionally remains `classic-original`.
